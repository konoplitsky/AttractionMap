# Attraction map

## Запустить сервер
В коре проекта выполнить команды
###  Сборка
``` bash
docker-compose build
```

### Запуск
``` bash
docker-compose up -d
```

В докере поднимается api и бд, чтобы просмотреть документацию к api необходимо перейти по url

```
http://localhost:10002/api
```

## Запустить клиент
```
cd web
```

### Установить зависимости
```
yarn
```

### Запустить локально

```
yarn dev
```
